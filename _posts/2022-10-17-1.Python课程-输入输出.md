### 7-2

```python
def calc():
    year = eval(input())
    if year <= 0 or len(str(year)) != 4 :
        print('ERROR!')
        return 0
    elif (year % 100 != 0 and year % 4 == 0 ) or ( year % 400 == 0):
        print(str(year)+'是闰年')
        return 0
    else:
        print(str(year)+'不是闰年')
        return 0
if __name__ == '__main__':
    calc()
```
